# Butyplast site


## What's included?

* Blueprints
* Flask-SQLAlchemy for databases
* mysql-connector
* Flask-WTF for forms
* Flask-Mail for sending emails

## Setting up

* Clone the repo
* $ git clone <https://github.com/Andreyinweb/butyplast.git>
* Create a virtual environment
* $ python3 -m venv venv
* $ source venv/bin/activate
* $ cd butyplastsite
* $ pip3 install -r requirements.txt
